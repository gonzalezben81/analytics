Regular Expressions (regex) in R

The following is a brief overview with examples you can use to learn regex in R. regex is a useful tool when you need to change text data or perform textual analysis.

Using grep in R

Here we can locate where a pattern is located utilizing the grep() function.

grep(pattern = "are",x = "hello how are you doing today")
## [1] 1

Using grepl in R

Using grepl() we can check to see if the pattern exists in the text. grepl() returns a logical value either TRUE or FALSE.

grepl(pattern = "are",x = c("are","you","there","are"))
## [1]  TRUE FALSE FALSE  TRUE

Using gsub in R

Using the gsub() function we can substitute words in text. gsub() will replace all of the matches in a string.

gsub(pattern = "you",x = "hello how are you doing today and who are you?",replacement = "they")
## [1] "hello how are they doing today and who are they?"

Using rexexpr or gregexpr in R

Utilizing regexpr we can find the position of the pattern match as well as the length of the patter match as well. This can come in hand with large text files that are not easily parsed by the human eye.
